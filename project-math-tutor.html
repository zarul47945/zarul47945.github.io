Full-Stack Web Project 2026

Math Tutor.

A deployed online mathematics classroom that connects public landing pages, teacher and student access, admin-controlled accounts, lesson sessions, worksheets and LiveKit room preparation.

Next.js + Supabase + LiveKit + TypeScript

A real tutoring platform, not only a landing page.

Math Tutor supports the full learning flow: visitors reach the live website, teachers and students sign in, admins manage accounts and assignments, lessons can be prepared for assigned students, and the live room brings video, shared work and worksheet activity into one place.

Project summary

2026
Online mathematics classroom

Math Tutor web platform

The project is built from the Math Tutor app source folder and deployed online. It focuses on making private mathematics lessons easier to organize, start and run from a browser.

  • Built a public Next.js website with student and staff login paths.
  • Implemented admin, teacher and student flows for controlled account access and lesson organization.
  • Used Supabase for authentication, profiles, assignments, sessions, worksheet records and room state.
  • Prepared LiveKit video-room access through a Supabase Edge Function so API secrets stay server-side.

How it works

01 / Admin

Account control

Administrators can create controlled teacher, student and admin accounts, then assign students to the correct teacher.

02 / Teacher

Lesson setup

Teachers see assigned students, create lessons, prepare worksheets and open active sessions for the learner.

03 / Student

Guided classroom

Students sign in, open prepared lessons, enter the live room and work through video, whiteboard and worksheet activities.

Main features

Role dashboards

Separate experiences for admins, teachers and students keep the platform organized around each user's responsibility.

Live classroom

LiveKit room preparation supports video lessons while keeping join-token generation on the server side.

Worksheet flow

Teachers can prepare worksheet questions or upload worksheet files, then students can answer during the lesson.

Shared tools

The room includes a shared timer, participant state, whiteboard drawing and synchronized lesson activity.
